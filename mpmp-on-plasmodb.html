Moving MPMP pathways to PlasmoDB

Phase 1. Get the site and data into a public format

This would ideally be a git repository with all the data, all the software that is currently used to run it, and instructions on how to run it.

The rest of the plan depends on what we can get here.

Phase 2. Host MPMP pages as they are

Two routes:

2a) Host the application

Need to check with Bob that it’s possible in general - would probably involve some dev/ops work to containerise the app.

2b) Turn MPMP into static content and hosting the pages as HTML read from disk

This is more for preservation if the previous option is impossible, since it would make any editing of the content very difficult.

MPMP gets crawled and preserved on archive.org, with snapshots going back from 2017, so if it were to go away tomorrow, this is always a lifeline.

Phase 3. Integrate the content with PlasmoDB content

Pathway content

Previously the project has tried to extract the content from the images, and building cytoscape drawings. This approach has not worked well enough.

A different approach is to integrate pathway drawings as interactive maps, with the functionality as for other pathways. Quick demo: https://wbazant.github.io/mpmp-plasmodb/
It could work better, but maybe still isn’t that good.

The added value compared to phase 2:

  • the overlays: painting enzymes, adding charts from RNASeq and other experiments
  • showing the pathway for other organisms with their EC inferred
  • some kind of overall cohesion (maybe)

Viewing data for P. falciparum, and searching through pathways are available on MPMP today. Enrichment analysis is already available, and the preservation can be done through phase 2.

Other

MPMP also involves a lot of annotation about the genes: publications about subcellular locations of proteins corresponding to the gene. They can be imported as cross-references.

There’s also some annotations for pathways - we can also have cross-references for pathways.

Some points raised in discussion

For other organisms, the pathways may be completely wrong. To address that, the MPMP pathways would only appear on PlasmoDB and UniDB, and there should be a disclaimer or some kind of information about what the data is.

Quality is lost when using orthologs to identify enzymes across species. MPMP maps ECs to specific genes in each pathway - this should be preserved for P. falciparum.
