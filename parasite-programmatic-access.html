API and programmatic access to ParaSite

Good for:

Automating website usage

  • Instead of genome browser: programmatically look for overlapping genes, search the flanking sequence, etc.

Hops in multi-step workflows

  • given a gene, get its orthologs, then check if they preserve a certain protein domain

Having a different data format

  • Get all the gene models as we structured them, instead of downloading the annotation file
  • Discover what data gets returned for endpoints that seem helpful, instead of looking through attributes in BioMart

When BioMart is better:

  • bulk retrieval of the data
  • everything that fits into “filters and attributes” framework

There’s an R package biomaRt if you need programmatic access.
